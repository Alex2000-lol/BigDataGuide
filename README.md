### 一、编程语言  
编程语言部分我是把我学的、用的都放在上面了，Java是基础，Scala是后续方便学习Spark和Flink，也是为了迎合以后发展需要，Python这门语言不会感觉确实是个遗憾，所以。。。我就都放上来了（哈哈），**前期有Java基础就可以了，其它的后面需要时再补**，上手也很快的。（别问我怎么学，问就是使劲拼命学 [ 吃瓜.jpg ]）
